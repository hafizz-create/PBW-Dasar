Pengertian Webite

Website adalah kumpulan halaman web yang saling terkait dan dapat diakses melalui internet, berisi informasi seperti teks, gambar, video, dan elemen lainnya, yang digunakan untuk berbagai keperluan seperti bisnis, pendidikan, hiburan, dan lain-lain.

Bahasa Pemrograman

– HTML (HyperText Markup Language)
    Struktur dasar halaman web digunakan untuk membuat elemen-elemen seperti teks, gambar, tabel, dan form.
– CSS (Cascading Style Sheets)
    Mengatur desain dan tampilan website, termasuk warna, layout, dan animasi
– JavaScript
    Menambahkan interaktivitas pada website, seperti animasi, validasi formulir, dan manipulasi elemen HTML secara dinamis..
– PHP (Hypertext Preprocessor)
    Bahasa backend yang populer untuk mengolah data dan menghubungkan ke database.
– Python
    Banyak digunakan untuk pengembangan backend berkat sintaks yang sederhana dan mudah dibaca.

Cara Website Bekerja

–Permintaan Data
    Ketika pengguna mengetik URL atau mencari sesuatu, perangkat mengirim permintaan ke server melalui jaringan.
–Rute Melalui ISP
    Permintaan melewati Internet Service Provider (ISP), yang menghubungkan perangkat ke internet.
–Server Merespons
    Server web mengirimkan data yang diminta kembali ke pengguna.
–Tampilan di Browser
    Data diterjemahkan oleh browser dan ditampilkan sebagai halaman web.
